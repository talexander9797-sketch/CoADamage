# CoADamage

A DRDamage-inspired theorycrafting addon for Project Ascension: Conquest of Azeroth.

## Current features

- Records direct, periodic, and ranged spell damage
- Separates normal and critical hit statistics
- Captures player stats and active buffs for each observation
- Provides `/coa stats`, `/coa inspect <spellID>`, and `/coa analyze <spellID>`
- Automatically groups observations into experiments when gear, buffs, or the target changes

## Experiment command

Use `/coa experiment` after dealing damage to view the current experiment ID, reason, and observation count.

## Installation

Copy the `CoADamage` folder into `Interface/AddOns`, then restart the game.
